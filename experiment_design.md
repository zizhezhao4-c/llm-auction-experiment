# 实验设计：大模型 Agent 的拍卖出价行为

> 课程：数字经济下的市场设计 · 第一次小组作业 · 选题①（拍卖/机制）
> 一句话：把 LLM 当成「拍卖参与者」，看它的出价是否符合博弈论预测，并量化偏离。

## 1. 研究问题（RQ）
- **RQ1（二价/Vickrey）**：给定私人估值 v，LLM 是否**如实出价**（b = v）？
- **RQ2（一价/First-price）**：LLM 是否会**压价**，且压价幅度是否趋近贝叶斯纳什均衡 b = v·(n−1)/n？
- **RQ3（影响因素）**：出价/偏离如何随**竞拍人数 n**、**人设（小白/商业/博弈论）**、**是否提示「想想最优策略」**而变化？
- **RQ4（理性边界）**：是否出现**过度出价**（b > v，二价中被占优、一价中中标即亏）？
- **RQ5（收入等价）**：从实测出价模拟拍卖，二价与一价的**期望收入**是否相等（收入等价定理）？

## 2. 理论基准（评分关键——所有结论都对照它）
设估值 iid ~ U[0, 100]，n 个竞拍者，风险中性：
| 机制 | 均衡出价 b*(v) | 备注 |
|---|---|---|
| 二价 / 英式 | **b\*(v) = v** | 占优策略，如实出价 |
| 一价 | **b\*(v) = v·(n−1)/n** | 线性压价，人越少压得越狠 |
| 期望收入（两者相等）| **(n−1)/(n+1)·V_max** | 收入等价定理 |

## 3. 自变量 / 处理（treatments）
- `mechanism` ∈ {second_price, first_price}（english 作为可选扩展）
- `n_bidders` ∈ {2, 3, 5}
- `persona` ∈ {naive 小白, business 商业经理, game_theorist 博弈论者}
- `framing` ∈ {rules_only 仅规则, reason_hint 提示先想最优策略}
- `value` v：每个 cell 随机抽 N 个（默认 30）∈ U[5, 100]
- `temperature`：主实验 = 0（可复现）；附加一组 temperature=1.0 看随机性（鲁棒性）

共 2×3×3×2 = **36 个 cell**，每 cell 30 个估值 → **1080 次调用**（deepseek-flash 成本很低）。

## 4. 固定 / 控制
- 每个实验条件**单独一次对话**（无上下文串扰）。
- **强制 JSON 输出**便于解析。
- 选项/措辞固定；对 framing 做 1–2 个 prompt 变体的稳健性检查。

## 5. 输出字段（CSV）
`run_id, timestamp, provider, model, mechanism, n, persona, framing, temperature, value, theory_bid, parsed_bid, bid_value_ratio, deviation(=b−theory), valid, refused, reason, raw_response`

## 6. 评价指标
- **二价**：如实出价率（|b−v|/v ≤ 5%）、过度出价率（b>v）、平均 b/v、回归 b=α+βv（期望 α≈0, β≈1）。
- **一价**：按 n 分组拟合 b=β·v，比较 β 与理论 (n−1)/n；R²。
- **收入等价**：用实测出价函数模拟 n 人拍卖，比较二价/一价期望收入与理论。
- **异质性**：如实率/压价幅度随 persona、framing 的变化。

## 7. 校验（killer comparison）
把实测出价散点叠加理论直线，报告斜率、截距、R²、平均偏离——一眼看出 LLM 在哪条机制上"懂规则"、在哪偏离。

## 8. 预期与局限
- 预期：二价较接近如实；一价**压价不足**（偏向 truthful，收入偏高）；博弈论人设 + 提示更接近理论。
- 局限：模型可能"背诵"拍卖理论而非真实行为；temperature=0 抹平个体差异；prompt 敏感；单一模型；无真实赌注。**报告需明确这是低成本预实验，不能等同真实人类行为。**
